Channels and units in Paradigma schemas

Within the paradigma project, some of the field types are further specialised to provide a better description of the data. These are described in the following sections.

Field: channels

Type: channel_type[]
Description: Describes the content of the data written. channel_type is specific to the Digital biomarkers for PD extension.

General types

Recommended unit

absolute_ms

Field: units

Type: unit_type[]

Description: Describes the format of the data written. unit_type is specific to the Digital biomarkers for PD extension.

unit_type

Description

relative_ms

Time (in milliseconds), relative to the start_iso8601, i.e., time elapsed since the start time of the recording.

difference_ms

Time (in milliseconds) elapsed since the previous sample.

absolute_ms

Absolute time in milliseconds, relative to Unix epoch.

probability

Probability values (0 to 1) indicating the likelihood of tremor activity for each sample.

boolean_num

Integer values (0 or 1) representing the true (1) or false (0) presence of an activity.

unitless

Numerical values without units.

g

Acceleration in g (meters per second squared / 9.81).

deg/s

Angular velocity in degrees per second.
