paradigma.load

Data loading module for ParaDigMa toolbox.

This module provides functions to load sensor data from various formats: - Raw data: TSDF (.meta/.bin), Empatica (.avro), Axivity (.CWA) - Prepared data: parquet, pickle, csv

Based on device_specific_data_loading tutorial.

Get list of data file paths without loading them.
